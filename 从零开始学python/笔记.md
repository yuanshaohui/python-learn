### 1.4linux终端命令

|       `ll`       |        当前文件夹下的文件        |
| :--------------: | :------------------------------: |
|      `pwd`       |           当前所在位置           |
|  `cd [目录名]`   |            进入该目录            |
| `touch [文件名]` | 打开这个文件，若没有则创建该文件 |
|  `mkdir [目录]`  |             创建目录             |
|  `rm [文件名]`   |            删除该文件            |
|     `clear`      |               清屏               |

#### 1.4.1查阅该命令的使用方式：

- `[命令] --help`:该命令的帮助

- `man [命令名]：`手册

  | 操作键  |      功能       |
  | :-----: | :-------------: |
  |  空格   |   翻到下一屏    |
  | enter键 |  一次 滚动一行  |
  |    b    |    回滚一屏     |
  |    f    |    前滚一屏     |
  |    q    |      退出       |
  | /world  | 搜索world字符串 |

#### 1.4.3`ls`命令的使用

|  `ls -l`   | 文件以列的形式展示 |

#### 1.4.4ls通配符的使用

| 通配符  |              含义              |
| :-----: | :----------------------------: |
|   `*`   |          代表任意字节          |
|  `？`   |          代表一个字节          |
|  `[]`   | 代表可以匹配字符组中的任意一个 |
| `[1-3]` |   代表当前字节，在这一范围内   |
